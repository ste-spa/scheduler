Vorwort

Dieser Simulator wurde in JavaScript, HTML sowie CSS entwickelt.
Er enthält vier implementierte Scheduling-Algorithmen.
Die Werte für die Variablen von Ankunftszeit, Priorität und der Laufzeit eines Prozesses werden zufällig generiert.
Nach der Erstellung von den Prozessen werden diese je nach Algorithmusart nach diesem Schema abgearbeitet.

Kurzanleitung für die Benutzung

  1. Wählen Sie den gewünschten Algorithmus aus, den Sie ausführen möchten.
  2. Geben Sie die gewünschte Laufzeit ein.
  3. Klicken Sie auf die Schaltfläche "Start" um den ausgewählten Algorithmus zu starten.
  4. Durch den optionalen "Continue-Modus" werden die Tasks automatisch erzeugt.
  5. Bei einem klick auf den Reset-Button werden die bestehenden Tasks gelöscht und die Statistik genullt.
Hinweis: Es werden maximal 50 Tasks generiert.
